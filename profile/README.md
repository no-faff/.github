# No Faff

Small, useful software. No accounts, no subscriptions, no bloat.

| | What it does |
|---|---|
| **InstallerClean** | Safely cleans orphaned files from C:\Windows\Installer — the hidden folder Windows never tidies up. Replaces PatchCleaner. |
| **Simple Browser Picker** | Pick which browser and profile opens every link. Silent routing rules, SafeLinks unwrapping, every Chromium and Firefox profile detected. |
| **Smart Auto Refresh & Notify** | Auto-refresh any browser tab on a timer with optional sound alerts when text appears or disappears. Chrome and Firefox. |
| **Simple Video Downloader** | Download video and audio from the browser via yt-dlp. Queue, history, filename templates. Firefox. |
| **Transcribe** | Local speech-to-text on your GPU using Whisper. No cloud, no uploads, no waiting. |

Everything is free, open source and MIT-licensed. No telemetry, no data collection.

If something here saves you time, a star on the repo or a cup of tea on [Ko-fi](https://ko-fi.com/nofaff) is always appreciated.
